  • Load Post

Load Post:

In load post the shipper can post the load. On the load post shipper puts the all details or information about the load and special instruction for the load. In Details, a shipper will tell about where is the pickup point or location and where the delivery would be made and also mention about timing delivery when the driver will pick it and drop the load. In special instruction shipper can comment what type of truck he needs.

For example: Pickup Truck, Dumper, Reefer, Dry van, Flatbed Truck, Box Truck, Trailer, Semi-Trailer, Tank Truck, Tata 407, Western Star and What is the length of truck.

Search Load:

By using this application the truck driver or any transport can search the load according to their needs. In the search load module driver can search the load according to their geographical location.

The main attributes are:

Origin: The pickup location

Destination: Delivery location in other word where he wants to go.

Truck type: Which Type of truck he needs/owned.

Length of the Truck: Define the truck type.

Weight: Define the weight of the load.

Start Date: Pickup date.

End Date: Drop/Delivery date.
